# Обзор проекта

## 1. Цель

Проект предназначен для демонстрации практической реализации темы «Линейная алгебра» в рамках первой контрольной точки:

- матрицы;
- определитель;
- СЛАУ.

## 2. Структура репозитория

```text
├── matrix_calculator.py         # ядро вычислений

```

## 3. Архитектура

### 3.1 Слои

1. **Математическое ядро** (`matrix_calculator.py`)
   - реализует операции;
   - валидирует размеры/ограничения;
   - возвращает `CalculationResult(steps, result)`.

2. **Пользовательский интерфейс** (`app.py`)
   - принимает ввод;
   - вызывает функции ядра;
   - показывает шаги, результат и ошибки.

3. **Тестовый слой** (`tests/`)
   - проверяет корректность вычислений и ошибок.

### 3.2 Принципы проектирования

- Ядро не зависит от GUI.
- Каждая операция является отдельной функцией.
- Исключения доменного уровня объединены в `MatrixError`.
- Вычисления сопровождаются журналом шагов (`steps`).

## 4. Поток выполнения

1. Пользователь вводит данные в UI.
2. UI парсит ввод в Python-структуры (`list` / `list[list]`).
3. Вызывается нужная функция ядра.
4. Результат и шаги отображаются в текстовом окне.
5. При ошибке показывается диагностическое сообщение.

## 5. Нефункциональные характеристики

- Зависимости: только стандартная библиотека Python.
- Переносимость: Linux/macOS/Windows (при наличии Tkinter).
- Расширяемость: можно добавлять новые операции в ядро и прокидывать кнопки в UI.
