# FAQ и диагностика

## Почему появляется сообщение «Матрица не должна быть пустой»?

Вы передали пустой список `[]` или некорректный формат.
Используйте минимум одну строку и один столбец: `[[1]]`.

## Почему операция сложения/вычитания не выполняется?

Матрицы должны иметь одинаковую размерность `m × n`.

## Почему не считается `det(A)`?

Определитель существует только у квадратной матрицы.

## Почему метод Крамера не работает?

Метод Крамера применим только при `det(A) != 0`.
Если `det(A)=0`, система либо несовместна, либо имеет бесконечно много решений.

## Что делать, если результат «не тот»?

1. Проверьте формат входа (`[[...], [...]]`, `[... ]`).
2. Проверьте порядок строк/столбцов.
3. Перепроверьте вектор `b`.
4. Сравните результат между тремя методами СЛАУ.

## Есть ли поддержка векторов и лабораторных работ 1–3?

В текущем состоянии репозиторий покрывает первую контрольную точку
(матрицы/определитель/СЛАУ). Для следующих контрольных точек функциональность нужно расширять.
