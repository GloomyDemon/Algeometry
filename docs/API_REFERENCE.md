# Справочник API (`matrix_calculator.py`)

## Общие типы

- `Matrix = List[List[float]]`
- `Vector = List[float]`
- `CalculationResult`
  - `steps: List[str]` — текст промежуточных шагов;
  - `result: object` — итог вычисления.
- `MatrixError(ValueError)` — доменная ошибка вычислений/валидности данных.

---

## Валидаторы и внутренние функции

### `_shape(matrix: Matrix) -> tuple[int, int]`
Проверяет, что матрица непустая и прямоугольная, возвращает `(rows, cols)`.

**Ошибки**
- `MatrixError("Матрица не должна быть пустой")`
- `MatrixError("У матрицы разные длины строк")`

### `_copy(matrix: Matrix) -> Matrix`
Глубокая копия по строкам.

### `_inverse(a: Matrix) -> Matrix`
Нахождение обратной матрицы через расширенную матрицу и метод Гаусса.

**Ошибки**
- неквадратная матрица;
- вырожденная матрица.

---

## Операции с матрицами

### `add_matrices(a, b) -> CalculationResult`
Складывает матрицы одинакового размера.

### `subtract_matrices(a, b) -> CalculationResult`
Вычитает матрицы одинакового размера.

### `multiply_by_scalar(a, scalar) -> CalculationResult`
Умножает каждый элемент матрицы на скаляр.

### `multiply_matrices(a, b) -> CalculationResult`
Умножает `A(m×n)` на `B(n×k)`.

**Ошибки**
- если число столбцов `A` не равно числу строк `B`.

### `transpose(a) -> CalculationResult`
Возвращает транспонированную матрицу.

### `determinant(a) -> CalculationResult`
Вычисляет определитель квадратной матрицы методом Гаусса с выбором ведущего элемента.

**Особенности**
- при нулевом ведущем элементе возвращается `0.0`.

### `rank(a) -> CalculationResult`
Вычисляет ранг методом приведения к ступенчатому виду.

---

## Решение СЛАУ

### `solve_slae_matrix_method(a, b) -> CalculationResult`
Решает систему по формуле `x = A^-1 * b`.

**Предусловия**
- `A` квадратная;
- `len(b) == n`;
- `det(A) != 0`.

### `solve_slae_cramer(a, b) -> CalculationResult`
Решает систему методом Крамера.

**Предусловия**
- `A` квадратная;
- `len(b) == n`;
- `det(A) != 0`.

### `solve_slae_gauss(a, b) -> CalculationResult`
Решает систему методом Гаусса (прямой и обратный ход).

**Предусловия**
- `A` квадратная (в текущей версии);
- `len(b) == n`;
- наличие единственного решения.

---

## Политика ошибок

Все ожидаемые математические и валидационные ошибки сигнализируются через `MatrixError`.
Рекомендуемое поведение клиента (UI/CLI):

1. перехватывать исключение;
2. показывать пользователю человекочитаемое сообщение;
3. не завершать приложение аварийно.
