# Руководство пользователя

## 1. Запуск

```bash
python3 app.py
```

После запуска откроется окно с полями ввода и набором кнопок операций.

## 2. Интерфейс

- **Матрица A** — обязательное поле почти для всех операций;
- **Матрица B или вектор b** — второе поле:
  - матрица `B` для матричных операций;
  - вектор `b` для СЛАУ;
- **Скаляр** — число для операции `k * A`;
- **Кнопки операций** — запуск команд;
- **Окно вывода** — шаги вычислений, результат, ошибки.

## 3. Формат данных

### 3.1 Матрица

Вводите в формате Python-списка списков:

```text
[[1, 2], [3, 4]]
```

### 3.2 Вектор

```text
[5, 6]
```

### 3.3 Скаляр

```text
2.5
```

## 4. Сценарии использования

### 4.1 Сложение матриц

1. Введите `A` и `B` одинакового размера.
2. Нажмите `A + B`.
3. Прочитайте шаги и результат.

### 4.2 Определитель

1. Введите квадратную матрицу в `A`.
2. Нажмите `det(A)`.
3. Внизу увидите шаги метода Гаусса и итог.

### 4.3 СЛАУ методом Гаусса

1. Введите матрицу коэффициентов в `A`.
2. Введите вектор свободных членов в поле `B/b`.
3. Нажмите `СЛАУ: Гаусс`.

## 5. Частые ошибки ввода

- Пропущены скобки/запятые;
- Непрямоугольная матрица (разные длины строк);
- Размерность `b` не совпадает с числом строк `A`;
- Попытка найти определитель неквадратной матрицы.

## 6. Как интерпретировать сообщения

- `Сбой или невозможность выполнить операцию` + текст — это ожидаемая диагностика.
- Приложение продолжает работу: исправьте входные данные и повторите команду.
