# Руководство разработчика

## 1. Локальный запуск

```bash
python3 app.py
```

## 2. Локальная проверка тестов

```bash
python3 -m unittest discover -s tests -v
```

## 3. Расширение функциональности

### 3.1 Добавление новой операции в ядро

1. Откройте `matrix_calculator.py`.
2. Добавьте отдельную функцию `operation_name(...) -> CalculationResult`.
3. Валидируйте входные данные через `_shape` и проверки размерностей.
4. Возвращайте:
   - список шагов в `steps`;
   - итог в `result`.
5. Для ошибок используйте `MatrixError`.

### 3.2 Подключение операции в UI

1. Импортируйте функцию в `app.py`.
2. Добавьте обработчик `do_<name>`.
3. Добавьте кнопку в список `operations`.
4. В обработчике вызывайте `self.show(...)` или `self.show_error(...)`.

### 3.3 Добавление тестов

1. Создайте/обновите тесты в `tests/test_matrix_calculator.py`.
2. Покройте:
   - позитивный сценарий;
   - граничный случай;
   - негативный сценарий с `MatrixError`.

## 4. Рекомендации по качеству

- Детерминированные функции без побочных эффектов в ядре;
- Понятные тексты ошибок для конечного пользователя;
- Малые функции с одной ответственностью;
- Явные проверки размеров перед вычислением.

## 5. Возможные улучшения

- Добавить поддержку прямоугольных систем в Гауссе (ранг, множество решений);
- Реализовать экспорт шагов вычислений в файл;
- Добавить CLI-режим (без GUI);
- Добавить property-based тестирование.
