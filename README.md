## 1. Назначение проекта

Приложение реализует функциональность **первой контрольной точки** курса «Алгебра и геометрия»:

- Матрицы;
- Определитель;
- СЛАУ.

Проект ориентирован на демонстрацию:

- математически корректных вычислений;
- промежуточных шагов вычисления;
- понятных сообщений об ошибках, включая математическую невозможность операции.

---

## 2. Что реализовано

### 2.1 Операции над матрицами

- Сложение матриц;
- Вычитание матриц;
- Умножение матрицы на число;
- Умножение матриц;
- Транспонирование;
- Вычисление определителя;
- Вычисление ранга.

### 2.2 Решение СЛАУ

- Матричный метод (`x = A^-1 b`);
- Метод Крамера;
- Метод Гаусса.

### 2.3 UI и взаимодействие

- Ввод исходных данных с клавиатуры;
- Запуск операций кнопками (мышью);
- Отображение промежуточных действий и результата;
- Вывод диагностик при ошибках ввода и невозможности вычисления.

## 3. Быстрый старт

### 3.1 Требования

- Python 3.10+
- Стандартная библиотека Python (дополнительные пакеты не требуются)

### 3.2 Запуск приложения

```bash
python3 app.py
```

### 3.3 Запуск тестов

```bash
python3 -m unittest discover -s tests -v
```

---

## 4. Как работать в интерфейсе

1. Введите матрицу `A` в поле «Матрица A».
2. Введите матрицу `B` **или** вектор `b` в поле «Матрица B или вектор b».
3. При необходимости введите скаляр в поле «Скаляр».
4. Нажмите кнопку нужной операции.
5. В нижнем окне получите:
   - шаги вычисления;
   - итоговый результат;
   - сообщение об ошибке (если есть).

### Примеры формата ввода

- Матрица: `[[1,2],[3,4]]`
- Вектор: `[5,6]`
- Скаляр: `2.5`
