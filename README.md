# Алгебра и геометрия — контрольная точка №1

Этот репозиторий содержит рабочий калькулятор для первой контрольной точки по темам:
- матрицы;
- определитель;
- СЛАУ.

## Реализовано

### Операции над матрицами
- сложение;
- вычитание;
- умножение на число;
- умножение матриц;
- транспонирование;
- определитель;
- ранг.

### Решение СЛАУ
- матричный метод;
- метод Крамера;
- метод Гаусса.

### Требования из ТЗ
- есть ввод исходных данных и запуск команды кнопкой (мышью);
- есть клавиатурный ввод исходных данных;
- выводятся промежуточные шаги и итог;
- есть сообщения о математической невозможности операции;
- есть сообщения об ошибках/сбоях.

## Запуск

```bash
python3 app.py
```

## Формат ввода
- Матрица: `[[1,2],[3,4]]`
- Вектор: `[5,6]`
- Скаляр: `2.5`

## Тесты

```bash
python3 -m unittest discover -s tests -v
```
